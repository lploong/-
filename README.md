# 🎮 Super Mario - Simplified

一个纯前端实现的简化版超级马里奥游戏，包含 3 个关卡，使用 HTML5 Canvas 绘制，无需任何外部依赖。

## 在线试玩

👉 [https://lploong.github.io/-/](https://lploong.github.io/-/)

## 关卡

| 关卡 | 主题 | 说明 |
|------|------|------|
| 1-1 | 经典地面关 | 草地、砖块、问号块、管道、旗杆 |
| 1-2 | 地下洞穴 | 狭窄空间，连续地面无坑 |
| 1-3 | 天空关 | 空中平台，有间隙需要跳跃 |

## 操作

| 按键 | 操作 |
|------|------|
| ← → | 左右移动 |
| ↑ / Space | 跳跃 |
| Space | 开始游戏 / 重新开始 |

## 特性

- 3 个完整关卡，流畅过渡
- 像素风格角色和场景渲染
- 板栗仔敌人，可踩扁击杀
- 问号块产出金币，砖块可打碎
- 粒子特效（金币、碎砖、得分）
- 平滑摄像机跟随
- HUD 显示分数、金币、生命数
- 5 条命，死亡可复活

## 本地运行

```bash
# 使用 Python
python -m http.server 8080
# 或使用任意静态文件服务器
```

浏览器打开 `http://localhost:8080` 即可游玩。

## 技术栈

- 原生 HTML5 + CSS3
- Canvas 2D 渲染
- 纯 JavaScript，零依赖
